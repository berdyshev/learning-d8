Building amazing searches with Search API

by drunken monkey

Search servers

  • database search
  • ApacheSolr search
 

ApacheSolr server security

  • Protect Solr server from outside access
  • Add hash to path, or
  • Block access with firewall, or
  • Use HTTP authentication

Index configuration

  • Field index configuration
    • adding fields to index
    • type of indexes
    • difference between string and fulltext
  • Workflow configuration
    • Display unpublish nodes
    • Index hierarchical data
    • etc

Display searches

  • Search pages (simple, quick, low flexibilty)
  • Search views
 

Facets

  • "facet API" + "Search facets"
  • Incredibly flexible
  • Any type of search (but only some backends)

Additional modules

  • Autocomplete module — suggestions for search fields based on results
  • Spellcheck module — adds optional spellchecking (only for Solr)
  • Saved searches module — allows users save their searches (notify new results, any type of search)
  • Search API Location — integrates with Geofield: filter, sort and facet by proximity
  • Search API multi-index searches — search by some indexes in the same time
  • Facet API Pretty Paths — pretty path for search results with filter criterias

Upgrading your JS to Drupal 8

by Theodore Biadala (nod_)

Changes in JS of Drupal 8

  • drupal_add_js() doesn't work any more, we should use now #attached
  • scripts[] doesn't work any more, we should use hook_library_info()
    
function my_module_library_info() {
    $libraries['myscript'] = array(
    'js' => array('script.js'),
    'dependencies' => array(array('core.ajax')),
    );
}
Issue #1996238: Replace hook_library_info() by *.library.yml file

Drupal 7 problems

  • jQuery 1.4.4
  • jQuery + drupal.js on all pages
  • core js breaks easily
  • contrib JS is not great

 

 

Drupal 8 solutions

  • Update all third party JS libraries
  • Declare all scripts dependencies
  • Strict mode & JSHint
  • Coding standards

New libraries

  • jQuery 2, CKeditor
  • Backbone, Underscore, Modernizr
  • Joyride (used in Tour module)

 

New APIs

  • Drupal.announce(text,priority) — for screen readers
  • Drupal.displace(broadcast) — viewport offset distances
  • Drupal.debounce(func, wait)
  • Drupal.dialog(element, options) — jQuery UI Dialog
 

New features: Responsive tables, Responsive images, Quick edit, many more...
